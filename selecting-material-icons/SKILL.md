## Icon Naming Patterns

Material icons follow these patterns:
- `{object}` → `home`, `settings`, `person`
- `{action}_{object}` → `add_circle`, `edit_note`
- `{object}_{action}` → `shopping_cart`, `account_circle`
- `local_{type}` → `local_shipping`, `local_offer`
- `{direction}` → `arrow_back`, `arrow_forward`

## JSON Search Tool

You can also use the companion script in this skill folder to query the metadata JSON:

- **Script**: `scripts/icon_search.py`
- **Purpose**: Quickly find icon names by keyword, collection-like label, or field-like label using the official `data.json` metadata.

### CLI Usage

```bash
# Search for icons by free-text query
python scripts/icon_search.py "shopping cart"

# Get suggestions for a collection-like label
python scripts/icon_search.py --collection "blog posts"

# Get suggestions for a field-like label
python scripts/icon_search.py --field "email address"

# Just get the top result (handy for scripting)
python scripts/icon_search.py --quick "user profile"
```

The script downloads and caches `icons-metadata.json` on first run, then reuses that local JSON file for fast subsequent searches.
